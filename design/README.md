# 阿言设计文档

这里保存阿言音乐播放器 MVP 的设计资料，作为后续实现前后端功能时的共同参考。

阿言第一版是一个本地单用户音乐播放器：浏览器前端调用 Go 后端 API，后端扫描用户指定的本地音乐目录，解析音频元数据并写入 SQLite，前端展示合并后的多目录歌曲库，并通过底部播放器播放本地音频流。

## MVP 关键词

- 浏览器前端
- Go 本地后端
- SQLite 媒体库
- 多目录累加扫描
- 递归扫描
- 忽略隐藏文件
- 底部唯一播放器
- 右侧扫描任务管理
