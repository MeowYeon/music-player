# 阿言设计文档

这里保存阿言音乐播放器的设计资料。当前实现以 [当前架构](./architecture.md) 为准；带版本号的文档是对应阶段的设计记录，用来理解功能演进和取舍，不一定完全等同于当前代码。

阿言当前版本是一个本地单用户音乐播放器：浏览器前端调用 Go 后端 API，后端管理媒体库、扫描本地音乐目录、解析音频元数据并写入 SQLite，前端展示歌曲库、媒体库、普通歌单、我喜欢、最近播放，并通过底部播放器播放本地音频流。
